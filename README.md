# SM64 Overall Rank Calculator Project

This is a front end project that allows users to see a combined overall leaderboard for SM64 speedruns

## Description

This is a purely HTML/CSS/Javascript based web app projects. It uses the speedrun.com REST api for get data for the leaderbaords of 120, 70, 16, 1, and 0 star for all 3 platforms for WiiVC, Console, and Emulator. It creates a "Combined" leaderboard for each category, combined in the sense that is combines the console, wiivc, and emulator times using offsets to fairly compare the times on different platforms. Then, using those combined leaderboars, it creates an overall leaderboard that is visually displayed and can be cycled through and each player's details can be viewed as well.

## What I personally learned from building this project
In this project, I built on my javascript fundamentals but the main skill I wanted to learn was using the fetch API and working with json data, and formatting that data to be displayed, which was really the bread and butter of this project of getting data, organizing, and displaying it for the user which I know is a foundational skill for front end development. I made use of javascript objects and also focused on organizing code in different files for better maintainability and making it so it was easier to actually follow the flow of the program in the code. 

## How to run the program

To run this app, you can either go to the github pages site here: https://katiek-dev.github.io/sm64-overall-rank-calculator/

OR you can clone the repo on your local machine and start it from the index.html file as well. All code is client sided javascript.
